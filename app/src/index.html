About the numbers. GitHub Copilot meters premium requests (per-model multipliers), not money. The "Premium requests" figure is consumption against your monthly allowance — not a dollar amount. Token counts are shown when Copilot exposes them; otherwise they read 0 and the view leads with request volume. All data is read locally and read-only; nothing leaves your machine except the Copilot CLI call you trigger.
